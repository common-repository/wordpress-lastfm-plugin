Simple including of LastFM API

== Description ==
If you have an LastFM Account, you can insert playlists and tagclouds in your posts very easy.
No flash or silvlight. Simply insert [lfm.getTopArtists username] in your post. Available functions
are descripted at the plugin option page.

Be carefull PHP 5 is needed!


== Installation ==

1. Upload `lastfm-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert your LastFM Api ID in the Plugin Options Site
1. Place [lfm.functionname username] in your posts
